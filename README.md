# MKTG4604 Value Generation Project

Welcome to the Value Generation Project (VGP) for MKTG4604. This repository contains sample code, prompts, and experiments that demonstrate how to design, prototype, and evaluate AI-enabled marketing concepts that create measurable value.

## What this project is
The VGP is a hands-on assignment where we explore how generative AI can support marketing strategy, content creation, and customer value delivery. The goal is to move from idea to working proof-of-concept using structured workflows, clear assumptions, and measurable outputs.

## What’s inside
- `SampleCode/` includes runnable examples for LLM workflows (LangGraph + Ollama) and prompt templates.
- `SampleCode/prompts/` contains prompt files used by the examples.

## How to use this repo
1. Review the sample code to understand the workflow patterns.
2. Adapt the prompts and state definitions to your own project idea.
3. Iterate on the workflow until outputs align with your value hypothesis.

## Suggested workflow
- Define the customer problem and value hypothesis.
- Translate the workflow into a multi-step LLM pipeline.
- Test, refine, and measure output quality.
- Document what worked and what didn’t.
