# Codex Installation Guide

## Download the Codex App

1. You can download the Codex App by OpenAI from here: https://chatgpt.com/codex
2. Click on `Download the App` button
3. Go to your `Downloads` folder where a `.dmg` file (for MacOS) or a `.exe` file (for Windows) would have been downloaded.
4. Install Codex using the Codex Installer (depends on your Operating System)
5. Go to this article and verify your Student status for Free $100 Credits in Codex. You must associate your ChatGPT Account with Codex for these credits: https://help.openai.com/en/articles/20001147-codex-credits-for-students-terms-of-service
6. There is a fair chance your student status will not be verified by SheerID based on the documents you provide. Go to the Get Help Section and fill out the form: https://support.sheerid.com/en-US/help-center/contact-form?
7. You should be verified within a day. Enjoy your 100$ Credits!


## Using Codex to Develop your App

1. Create a Repository on Github and clone the empty Repository to your PC using `git clone https://www.github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME`
2. Open Codex and under the Threads tab, select the `Add Directory` Icon.
3. Select the newly created Repository from File Browser. It will open a ChatGPT Style UI where you can talk to it.
4. Select the `VS Code-Open` button on the top bar to open your project in VS Code.
